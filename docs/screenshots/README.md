# Screenshots

This directory contains the public proof screenshots used by the README and demo review flow.

The sequence is intentionally ordered around the institutional RFQ workflow:

1. `01-new-rfq-ticket.png` shows dealer selection intelligence, recommended counterparties, inventory axes, and simulated market tape context before an RFQ is sent.
2. `02-live-rfq-blotter.png` is the visual centerpiece: live dealer liquidity, quote status, best quote, latency, and execution actions.
3. `03-trade-analytics.png` shows best-execution proof: cover price, selected quote rank, slippage, spread paid, missed savings, and tape benchmarks.
4. `04-audit-log.png` shows compliance evidence: UTC event timestamps, event types, actors, entity IDs, and payload details.
5. `05-dealer-performance.png` shows counterparty analytics: RFQ volume, quote count, win rate, average spread, and latency.

These are static product screenshots for repository review. The YouTube demo linked from the README shows the same workflow as a narrated walkthrough.
