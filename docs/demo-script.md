# Demo Script

Open with:

> This is BondWorks Lite, a simulated fixed-income RFQ execution workstation.

1. Start on New RFQ, select `UST 4.25 2036`, BUY, `$5MM`, seven dealers.
2. Point out recommended dealers and inventory axes.
3. Submit RFQ.
4. On Live RFQ Blotter, narrate quote competition: CITI often best, JPM fastest, BARC slow/expired, DB no response.
5. Execute best quote, or intentionally execute non-best quote to show best-execution rationale.
6. Open analytics: selected quote rank, cover price, tape VWAP, slippage, spread paid, missed savings, latency.
7. Open Audit Log: RFQ created, sent to dealers, quote received, rationale submitted, trade created, analytics generated.
8. Flash architecture diagram and SDK snippet.
